## MTS Parser
Получи информацию о всех тарифах и подписках от МТС, нажав на одну кнопку. Гигабайты, минуты, цены и тарифные плюшки - все, что нужно, чтобы выбрать свой идеальный тариф!

### Requirements
- JDK 19
- Gradle 7
- Firefox

### Stack
- Java 19
- Gradle 7
- Spring Boot, Spring Web, Spring Data JPA
- Jsoup
- Selenium
- FlyWay
- PostgreSQL
- Lombok
- WebDriverManager


### Build application
```
make build
```

### Start application
```
make start
# http://localhost:8080
```
